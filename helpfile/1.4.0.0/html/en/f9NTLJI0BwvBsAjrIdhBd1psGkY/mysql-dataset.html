TYPE MySQL_DataSet : STRUCT

This Structure contains the response data of a successfull executed SELECT statement in clean format. To show the selected Data in a readable format, use the “MySQL_GetStringValue” to access and convert raw data into readable String.

InOut:
Name Type Comment
uiColumnCount UINT Number of Coulumns in a Dataset
arrColumns ARRAY [0..gc_MySQL_dwMaxColumns] OF MySQL_CellFormat Detailed Columns Informations
uiRowCount UINT Number of Rows in a Dataset
arrRows ARRAY [0..gc_MySQL_dwMaxRows] OF ARRAY [0..gc_MySQL_dwMaxRowSize] OF BYTE raw format of selected rows
